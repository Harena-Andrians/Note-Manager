# React notes manager

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/votre_nom_utilisateur/note-manager.git
   cd note-manager

## Getting Started

### Install the dependencies:
```bash
npm i
npm start
```
### Start json-server to simulate an API:
```bash
npm run json-server
```
### Start the React app in a new terminal:
```bash
npm start
```
